Running ToolsUI

Download and run locally

You can download the toolsUI.jar from:

  • the netCDF Download page, or
  • the netCDF-Java documentation page

You can then run the ToolsUI application using a command similar to this:

java -Xmx1g -jar toolsUI.jar

Check that netCDF-Java can open your dataset

The netCDF-Java library can read data from a wide range of scientific data formats (more details here). To check that your dataset can be read by netCDF-Java, use the "Viewer" tab in ToolsUI. Either cut-and-paste a URL or file path into the "dataset" field or click on the folder icon and select a file. (If you cut-and-paste, you will need to hit "Enter" to open the dataset.)

Check that netCDF-Java recognizes your dataset as gridded data

The netCDF-Java library can recognize various kinds of gridded datasets (e.g., a netCDF-CF compliant gridded file or GRIB data). To check that your dataset is recognized as gridded data, use the "FeatureType -> Gridded" tab in ToolsUI. Either cut-and-paste a URL or file path into the "dataset" field or click on the folder icon and select a file. (If you cut-and-paste, you will need to hit "Enter" to open the dataset.)

If the dataset is recognized as containing gridded data, the ToolsUI window will show three tables containing (from the top table down) gridded fields, coordinate systems, and axes. E.g.,
